Política de Privacidade - Versão Beta

1. Finalidade

Esta versão beta da aplicação Saltflow é destinada exclusivamente para testes internos pela equipe de desenvolvimento.

2. Dados Coletados

Para funcionalidade da aplicação, coletamos:

  • Nome, email e username para criação de conta
  • Foto de perfil (opcional)
  • Data de nascimento (para verificação de idade)
  • Informações de qualificação profissional
  • Cidade e nacionalidade

3. Uso dos Dados

Os dados são utilizados apenas para:

  • Testes de funcionalidades da aplicação
  • Identificação de usuários na plataforma
  • Melhoria do produto antes do lançamento oficial

4. Armazenamento

Os dados são armazenados em banco de dados seguro e serão excluídos após o período de testes.
